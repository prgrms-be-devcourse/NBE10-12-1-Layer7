# ☕ Layer7

원두 상품 주문 및 관리 시스템

부트캠프 협업 프로젝트로 진행된 웹 서비스입니다.

사용자는 원두 상품을 조회하고 주문할 수 있으며, 관리자는 주문 상태를 관리할 수 있습니다.
